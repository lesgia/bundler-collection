# Bundler Collection

## Introduction
This repository is a learning project focused on bundling techniques and tools.

## Features
- Collection of different bundlers and their configurations.
- Examples of usage for each bundler.

## Technology Stack
- JavaScript
- Various bundling tools (Webpack, Parcel, etc.)

## Usage
To explore the different bundlers, clone the repository and check the examples:
```bash
git clone https://github.com/lesgia/bundler-collection.git
cd bundler-collection
```
